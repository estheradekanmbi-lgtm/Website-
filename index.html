Corporate Blueprint Cohort 1 – Learner Performance Analysis

End-to-end data analytics project: from raw data to actionable insights

View GitHub View Dashboard Download Report
Overview Dashboard
Exploratory Analysis
Key Insights
📋 Overview ▾

This end-to-end analytics project analyses learner performance data from the Corporate Blueprint Cohort 1 training programme. The goal was to evaluate learner engagement, completion rates, and performance metrics to help programme administrators make data-driven decisions about curriculum design and learner support.

🔍 Business Problem ▾

The training coordinator needed answers to: Which learners are at risk of dropping out? Which modules have the lowest completion rates? What is the average score distribution across cohorts? How does engagement correlate with final scores? Without clear answers, resources were being misallocated and at-risk learners weren't getting early interventions.

🗂 Dataset Preparation ▾

Raw data was provided in Excel format with 8,240 rows spanning learner IDs, module scores, attendance, assignment submissions, and final results. Data cleaning involved: removing 127 duplicate entries, handling 43 missing score values via median imputation, standardising date formats, and normalising inconsistent category labels (Pass/PASS/pass → Pass).

📈 Exploratory Analysis (EDA) ▾

EDA was performed in both Excel (Pivot Tables, conditional formatting) and Python (Pandas, Matplotlib). Key analyses included score distribution histograms, correlation heatmaps between attendance and final scores, module-level completion rate comparisons, and learner segmentation by performance quartile.

📊 Power BI Dashboard ▾

An interactive Power BI dashboard was built with 4 report pages: Executive Summary (KPI cards, trend lines), Module Performance (bar + waterfall charts), Learner Risk Analysis (scatter plots, conditional highlighting), and Score Distribution (histograms, quartile breakdown). DAX measures were written for completion rate, pass/fail ratio, and average weighted score.

💡 Key Insights ▾
  • Module 3 had the lowest completion rate (61%) — content complexity likely too high for early-stage learners
  • Learners with attendance below 70% had a 3x higher risk of failing the final assessment
  • Top 25% of learners completed assignments 48 hours ahead of deadlines on average
  • Overall cohort pass rate was 78.4%, exceeding the 75% programme target
  • Three learner clusters identified: High-performers, At-risk, and Disengaged — enabling targeted support
