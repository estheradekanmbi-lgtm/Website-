Featured Case Study

Corporate Blueprint Cohort 1
Learner Performance Analysis

An end-to-end analytical investigation into learner engagement, performance drivers, and programme effectiveness.

Tools Used Excel · SQL · Power BI
Dataset Size 500+ Learner Records
Project Type Business Intelligence
Domain EdTech · HR Analytics
01

Business Problem

The Corporate Blueprint programme needed visibility into which learners were at risk of dropping out, which modules had the highest disengagement, and what factors predicted completion — enabling programme managers to intervene early.

02

Dataset & Sources

Cohort enrolment records, module assessment scores, attendance logs, assignment submission timestamps, and facilitator feedback — consolidated across multiple Excel sheets into a unified data model.

03

Data Cleaning Process

Removed 47 duplicate entries, standardised date formats across three source files, handled 12% missing assessment scores using median imputation per cohort group, and validated referential integrity between learner IDs and submission records.

04

Analysis Methodology

Segmented learners by performance quartile, applied attendance frequency analysis, calculated module completion rates, and used correlation analysis to identify the top predictors of programme success — including early assignment submission and peer session attendance.

05

Key Insights

  • Learners who submitted assignments within 48 hours of release were 3.2× more likely to complete the programme.
  • Module 3 had the highest drop-off rate at 28% — coinciding with the highest cognitive load week.
  • Peer-led session attendees outperformed non-attendees by an average of 22 percentage points.
  • 61% of at-risk learners could be identified by Week 2 using only 3 behavioural signals.
06

Recommendations

  • Deploy an early-warning system triggering facilitator outreach when a learner misses Week 2 signals.
  • Restructure Module 3 to reduce cognitive overload by splitting content into two shorter segments.
  • Increase incentives for peer-session attendance through certification recognition.
  • Introduce an automated weekly dashboard visible to both learners and programme managers.
