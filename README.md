# Esther Adekanmbi – Portfolio Website

    Premium single-page portfolio. HTML5 + CSS3 + Vanilla JS. No frameworks, no dependencies.
    Ready to deploy to GitHub Pages immediately.

    ## 🚀 Deploy to GitHub Pages

    1. Create a GitHub repo (e.g. `estheradekanmbi.github.io`)
    2. Upload ALL files maintaining the folder structure
    3. Go to **Settings → Pages → Source → main branch / (root)**
    4. Live at `https://estheradekanmbi.github.io`

    ## 📁 File Structure

    ```
    /
    ├── index.html
    ├── style.css
    ├── script.js
    ├── README.md
    └── assets/
      ├── images/
      │   └── placeholder.jpg      ← Replace with your real headshot
      └── resume/
          └── Esther_Adekanmbi_CV.pdf  ← Replace with your real CV
    ```

    ## ✏️ Customise

    | What | Where |
    |------|-------|
    | Your photo | Replace `assets/images/placeholder.jpg` with a square headshot |
    | Your CV | Replace `assets/resume/Esther_Adekanmbi_CV.pdf` |
    | GitHub link | Search `href="#"` in index.html → update GitHub URL |
    | LinkedIn link | Same — update LinkedIn URL |
    | Email | Replace all `esther@email.com` instances |
    | Color | `style.css` → `--primary: #7C3AED` |
